ChippyCal Development

Built step by step with real carpenter feedback.

ChippyCal is not meant to be a one-and-done calculator. The goal is to keep improving it with real site feedback, practical workflows, and useful features that save time on the job.

Current focus

Version 1 is focused on getting the core tools right before adding too much extra. The aim is speed, clarity, and practical value on real jobs.

What is in focus now

  • Metric and imperial conversions
  • Stud Mode / framing layout
  • Cut lists and workflow tools
  • Cleaner site-friendly calculator workflow

Why this matters

The goal is to remove friction from everyday site calculations so the app becomes something you can rely on quickly without digging through menus or fighting a standard calculator.

Coming next

Once version 1 is solid, future updates will build on the core workflow instead of adding random features for the sake of it.

Planned next features

  • Material estimator
  • Timber quantity calculations
  • Saved jobs and saved walls
  • More construction-focused tools

Community input

The app will continue to be shaped by user feedback. Feature suggestions, testing, and future voting will help decide what gets built next.
